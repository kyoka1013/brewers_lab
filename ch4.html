Chapter 4

豆を知る

同じ淹れ方でも、豆が変われば味は変わる。逆に、豆を見て「どう淹れるか」を予測できるようになると、初めて触る豆でも最初の一杯が遠回りしない。

1. 産地で何が決まるか

産地は「気候・標高・品種・精製・伝統」のすべてを束ねたラベル。代表産地と典型像を持っておくと、未知の豆に出会ったときの仮説の出発点になる。

産地典型像抽出方針の出発点
エチオピア（イルガチェフェ、シダモなど）フローラル、シトラス、紅茶様。ヘイルーム品種。浅煎り中心、92〜94℃、明るさを引き出す方向
ケニア強い酸（ブラックカラント、トマト）、ジューシー、SL28/34浅〜中浅、温度高め、酸の質を活かす
ルワンダ・ブルンジクリーン、紅茶様、シルキーボディ浅〜中浅、エチオピア寄りの設計
コロンビアバランス、ナッツ、キャラメル、品種多様中煎り基準、汎用性高い
ブラジルナッツ、チョコ、低酸、安定中〜中深、ボディ重視。ブレンドベース
グアテマラチョコ、スパイス、火山土壌の複雑さ中煎り、しっかり目の抽出
コスタリカクリーン、ハニー精製の名産地精製で味が変わる。中煎り中心
パナマ（ゲイシャ）強烈な花香、ジャスミン、ベルガモット浅煎り、抽出を引き出しすぎないバランス
インドネシア（マンデリン）アーシー、スパイス、低酸、重いボディ中深〜深、コクを引き出す方向
イエメン独特の野性味、発酵感、ドライフルーツ浅〜中浅、ナチュラル多。設計が読みづらい個性派
この表は「典型像」。同じ国でも農園・品種・精製で大きく振れます。「ケニア=酸が強い」は出発点に過ぎず、ロースターの設計でいくらでも変わる。

2. 品種で何が決まるか

アラビカ種の中の代表的な品種と、その典型キャラクター。

品種キャラクター備考
ティピカベース。クリーン、繊細多くの品種の祖
ブルボン甘さと丸み派生品種多数
カトゥアイ / カトゥーラブルボン由来、生産性重視中南米で広く栽培
SL28 / SL34ケニア独自、強烈な酸とジューシーさケニアの代名詞
ゲイシャ花香、ベルガモット、特異な明るさパナマで開花した品種
パカマラ大粒、複雑、エルサルバドル発競技でも人気
ヘイルーム（エチオピア在来）多様な野生品種の総称名前のないキャラクター
カスティージョ / カトゥンビータ耐病性重視のコロンビア品種生産性とのトレードオフ議論あり

3. 精製方法と味の傾向

収穫した果実をどう処理するかで、味は大きく変わる。同じ品種でも別物に感じることがある。

精製工程の要点味の傾向
ウォッシュト（水洗式）果肉除去 → 水洗発酵 → 乾燥クリーンで酸が明瞭。品種・産地の素性が出やすい。
ナチュラル（非水洗）果肉ごと天日乾燥果実味・甘さ・発酵感が強い。ブルーベリーやワイン様。
ハニー / パルプドナチュラル果肉除去後、ミューシレージ（粘質）を残して乾燥中間。甘さとクリーンさのバランス。White / Yellow / Red / Black で残量が違う
アナエロビック（嫌気発酵）密閉容器で酸素を断って発酵強い個性、独特のフレーバー（シナモン、トロピカル、ワイン）
カーボニックマセレーションワイン由来。CO2環境で発酵さらに個性的、フルーティ強調
スマトラ式（湿式脱穀）マンデリン特有。半乾きで脱穀アーシー、スパイス、低酸、重いボディ

4. 焙煎度で抽出方針が変わる

焙煎は豆を化学的に作り替える工程。抽出設計は焙煎度を前提に組み立てる。

焙煎度豆の状態典型味抽出方針
ライト〜シナモン（浅煎り）豆が硬く密度高い。成分が出にくい強い酸、フルーツ、フローラル細め・高温(93-96℃)・長め。「抽出を助ける」方向
ミディアム（中浅煎り）1ハゼ後明るい酸＋甘さ中細・92℃前後・標準時間
シティ（中煎り）飲用域の中心バランス、ナッツ、チョコ中・90-93℃・標準
フルシティ（中深煎り）2ハゼ直前チョコ、コク、控えめな酸中〜やや粗・88-91℃
フレンチ〜イタリアン（深煎り）多孔質、脆い、油分浮く苦味、スモーキー、コク粗・83-90℃・短め。「出すぎを抑える」方向

5. 鮮度と保存

  • 飲み頃：焙煎日からおよそ 3日〜2週間。豆により幅あり。直後はガスが多く抽出が乱れる。
  • 保存：密閉容器、冷暗所。冷凍は議論あり（取り出し時の結露問題）。
  • 粉：挽いた瞬間から急速劣化。淹れる直前に挽くのが原則。

6. 豆を見て予測する練習

新しい豆を入手したら、淹れる前に以下を口に出すと「予測」の筋肉が育つ。

  1. 産地・品種・精製・焙煎度を確認
  2. 「この組み合わせなら、たぶん酸は◯◯系、甘さは◯◯、ボディは◯◯」と予測
  3. 抽出方針の出発点を決める（挽き目・温度・時間）
  4. 淹れて答え合わせ（Ch.3で記録）
  5. 外れた点を Claude に「なぜ外れたか」相談

Claudeを豆の専門家にする
