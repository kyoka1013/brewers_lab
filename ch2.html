Chapter 2 · Sensory Language

味の言語化トレーニング

「美味しい」「美味しくない」で止まると、抽出の改善先が見えない。SCA Coffee Tasters Flavor Wheel と Q Grader の評価軸を、自分の感覚に翻訳する技術を身につける。

5軸の正体 感覚マッピング 記述ビルダー フレーバーホイール フレーバー辞書 抽出タイムライン 味覚キャリブレーション ネガティブ解剖学 Claudeに相談

1. 5軸の正体 — Q Grader はここを見る

SCA カッピングフォームを日常用に圧縮した5軸。それぞれに「強さ」だけでなく「質」を見る癖をつけると、抽出の判断材料が一気に増える。

①

酸質（Acidity）

Q-Grader View

「酸っぱい／酸っぱくない」ではなく、何の酸かを見る。コーヒーに含まれる酸は10種類以上あり、それぞれ感じる場所と質が違う。

酸の種類感じ方食材で言うと
クエン酸明るく、舌の両脇に鋭く立つレモン、ライム、グレープフルーツ
リンゴ酸丸く、フルーティでジューシー青リンゴ、洋梨、白ブドウ
酒石酸キレと余韻が長い赤ブドウ、ワイン
リン酸シュワッと弾ける感覚炭酸、コーラ
乳酸柔らかくクリーミーヨーグルト、発酵バター
酢酸ピリッと尖る（多すぎると欠点）酢、過発酵
Q
Q-Grader Tip: 「Acidity（酸の強さ）」と「Acid Quality（酸の質）」は別評価。強くても質が低ければ減点。一杯から複数の酸を識別できると評価語彙が一段深まる（"レモンのクエン酸とリンゴ酸の調和"のように）。エチオピア・ケニアは強く明るい酸、コロンビアは丸い、ブラジルは穏やか、というのは品種×精製×焙煎の総合結果。
②

甘さ（Sweetness）

Q-Grader View

コーヒーには糖類はほぼ残っていない。だが「甘く感じる」のはメイラード反応物・カラメル化生成物・有機酸とのバランスが脳に「甘い」と認識させているから。「砂糖の甘さ」より「余韻に残るどんな甘さか」で見る。

甘さのタイプ食材で言うと
シュガー系白砂糖、メープル、ブラウンシュガー
ハニー系蜂蜜、メープルシロップ
ドライフルーツ系レーズン、デーツ、いちじく
キャラメル系キャラメル、ヌガー、トフィー
チョコレート系ミルクチョコ、ダークチョコ
果実糖系熟した桃、メロン、マンゴー
Q
Q-Grader Tip: SCA カッピングでは Sweetness は uniformity と同じく "ある/ない" の二値的評価（10点満点で出る/出ない）。だが日常評価では「どんな甘さがどれくらい持続するか」を見るほうが抽出改善に役立つ。甘さが弱い＝抽出不足の最重要シグナル。
③

苦味（Bitterness）

Q-Grader View

苦味は2種類ある。由来で区別するのが大事。

由来性質味わい
ロースト由来焙煎の化学変化ダークチョコ、カカオ、香ばしい苦味
過抽出由来抽出のし過ぎ渋み、舌が乾く、後味が悪い
Q
Q-Grader Tip: カフェイン由来の苦味は実は全体の10%程度。多くはクロロゲン酸ラクトン（焙煎進行で増加）とフェニリンダン（深煎りで増加）由来。「ロースト苦味は心地よく、過抽出苦味は不快」と切り分けると、抽出を直すべきか焙煎度を変えるべきかが見える。渋み（astringency）は苦味とは別軸で、過抽出やタンパク質残留が原因。
④

ボディ（Body / Mouthfeel）

Q-Grader View

触覚的な評価軸。重さ・厚み・テクスチャ。SCA では「Heavy / Medium / Light」と「Smooth / Rough」の二次元で評価する。

口当たり食材で言うと
Watery（水様）緑茶、水
Tea-like（紅茶様）紅茶、ダージリン
Juicy（ジューシー）果実ジュース、果肉感
Creamy（クリーミー）牛乳、ラテ
Syrupy（シロップ様）蜂蜜、ハニー
Velvety（ベルベット）滑らかで上質
Q
Q-Grader Tip: ボディは主に油分・微粉・コロイド粒子で決まる。フィルター素材でほぼ上限が決まる：金属フィルター＞アバカ＞紙（白）の順でボディが増す。注ぎ方では「撹拌が強い」「総時間が長い」とボディが増す傾向。深煎り豆は油分が多いので元々ボディが乗りやすい。
⑤

余韻（Aftertaste / Finish）

Q-Grader View

飲み込んだ後、何が・どれだけ・どんな質で残るか。余韻の質が、スペシャルティとコモディティを分けると言っても過言ではない。

余韻のタイプ意味
Clean & Short後味がスッと消える。クリーン。
Sweet & Long甘さが伸びて続く（理想）
Floral / Tea-like Finish花や紅茶の余韻が残る
Dry / Astringent渋く乾く（過抽出/欠点）
Harsh / Burnt苦く焦げ感（過焙煎/過抽出）
Q
Q-Grader Tip: Aftertaste は SCA フォームで独立評価項目（10点）。飲み込んでから「7〜15秒の間」に何が残るかを意識する。慣れていないと「終わった味」を見逃しがち。冷めてから余韻だけ評価する練習も有効。

6. 味の抽出タイムライン — The Extraction Curve

コーヒー成分は一斉に溶け出すのではなく、溶けやすい順番がある。下のスライダーを動かすと、抽出をどこで止めたかによって、カップに残る味の構成がどう変わるかが見える。Ch.3 のレシピ開発で「挽き目を細かく＝抽出を進める」と聞いたとき、何が増えて何が消えるかをここで腹落ちさせる。

序盤酸味・果実味
中盤甘さ・ボディ
終盤苦味・渋み
抽出開始 30% 65% 抽出完了

適正バランス

序盤の酸と中盤の甘さがバランスし、終盤の苦味はまだ穏やか。SCAゴールデンカップ的にはこのあたり（EY 18〜22%相当）。

序盤：酸・果実
50%
中盤：甘さ・ボディ
35%
終盤：苦味・渋み
10%
Q
Q-Grader Tip: これは「成分の溶解順序」を理解するためのモデル図。実際には序盤・中盤・終盤が完全分離するわけではなく、すべて並行して進行している。だが「後ろの成分ほど後から強く出る」という大原則は実務的に有効。抽出を止めるタイミング（挽き目・湯温・時間）で、どの成分まで取り込むかが決まる。Ch.5 で扱う EY（抽出率）も基本的にこの曲線の位置を数値化したもの。

7. 自宅でできる！味覚キャリブレーションガイド

口の中に「これがクエン酸の感覚」「これが渋みの正体」という基準サンプルを持っていると、コーヒーを飲んだ瞬間の判定が一気に速く正確になる。SCA Sensory Skills の校正手法を、家にあるもので再現できる形にしました。週末に20分、ぜひ。

LV.1
🍋

レモン水でクエン酸を覚える

Citric Acid Calibration
明るく鋭い、舌の両脇に立つ酸

用意するもの

  • レモン汁 小さじ1（5g程度）
  • 水 100ml（常温）
  • ガラスのカップ

手順

  1. 水にレモン汁を加えて混ぜる
  2. 少量を口に含み、舌の両脇で受け止める
  3. 飲み込まずに5秒、酸の位置と質を観察
  4. 飲み込んでから余韻のキレ・長さを観察
こう感じたら正解 舌の両脇に鋭く立ち、唾液が湧く。後味は比較的短くキレる。これがクエン酸の典型像。ケニア・エチオピア浅煎り・コロンビア浅煎りで頻出する「明るい酸」。
LV.2
🍎

リンゴジュースでリンゴ酸を覚える

Malic Acid Calibration
丸く、ジューシーで持続性のある酸

用意するもの

  • 100%リンゴジュース 30ml（果汁100%表記のもの）
  • 水 70ml
  • ※ レモン水と飲み比べると違いがクッキリ

手順

  1. 水でリンゴジュースを薄め、口に含む
  2. クエン酸（レモン水）と違う「酸の位置」を意識
  3. 飲み込んでから後味の長さに注目
  4. もう一度レモン水と交互に。違いを言語化
こう感じたら正解 レモンより舌の前方〜中央で丸く感じ、後味に甘さが伸びる。これがリンゴ酸の典型像。コロンビア・コスタリカ・エルサルバドルの中浅煎りでよく出会う。
LV.3
🍵

放置紅茶で渋み（Astringency）を覚える

Astringency Calibration
タンパク質を縮ませる「触覚」の感覚

用意するもの

  • 紅茶のティーバッグ 1個
  • 熱湯 200ml
  • タイマー（スマホで可）

手順

  1. 熱湯にティーバッグを入れ、10分間放置する
  2. 少し冷ましてから一口含む
  3. 口の中の粘膜がキュッと縮む感覚に注目
  4. これが「苦味」ではなく「触覚的な収れん」だと確認
こう感じたら正解 舌や頬の内側が乾いて縮む感覚。これは味覚ではなく触覚。コーヒーで「渋い」と感じる時の正体。過抽出のサインとして最重要なので、この感覚を体に入れておくと診断速度が桁違いになる。
Q
Q-Grader Tip: SCA の Sensory Skills では、純粋なクエン酸・リンゴ酸・酒石酸を水に溶かした標準液でこれと同じ訓練をする。家でやるなら身近な食材で十分。3つを2週間隔週で繰り返すと、コーヒーを飲んだ時に「これはクエン酸ベース」「ここに渋みが混じってる」と瞬時に切り分けられるようになる。これが Ch.3 のレシピ開発で「次に何を直すか」の判断速度を決める。

8. ネガティブフレーバー解剖学 — Defect & Extraction Errors

初学者が混同しがちな3組のペアを解剖する。正しく区別できれば、Ch.3 でどのパラメータを動かすべきかが自動的に決まる。タップで展開。

苦味VS渋み
舌で感じる味覚 vs 粘膜で感じる触覚
▼
苦味（Bitterness）味覚

舌の奥で感じる、5基本味の一つ。ロースト由来であれば心地よい苦味（ダークチョコ、カカオ、香ばしさ）として機能する。

  • 主因はクロロゲン酸ラクトン・フェニリンダン（焙煎度で増加）
  • カフェイン由来は実は全体の1〜2割程度
  • 深煎り豆では「特徴」として歓迎される
渋み（Astringency）触覚

口の中の粘膜全体がキュッと縮む感覚。味ではなく触覚。タンニン様物質が口腔タンパク質と結合して起きる。

  • 主因は過抽出による高分子物質の溶出
  • 粒度分布の広さ（微粉混入）も原因
  • 不快で長く尾を引く。心地よさはない
Ch.3で動かすべきパラメータ 「苦味は強いが心地よい」なら何もしなくてOK（焙煎度の個性）。渋みが出ているなら過抽出のサイン。 挽き目を1段粗く→総時間を10〜20秒短く→湯温を2℃下げるのうち1つだけ動かして検証。
酸味（Acidity）VS酸っぱさ・酢様（Sour）
フルーティで明るい酸 vs 不快に尖る欠点
▼
酸味（Acidity）スペシャルティ

レモン・リンゴ・ベリーのようなフルーツを想起させる明るい酸。クエン酸・リンゴ酸・酒石酸が主役。甘さと共存するのが特徴。

  • SCAカッピングでは高評価対象（Acidity 8点以上）
  • 余韻に甘さや果実感が伸びる
  • 「明るい」「ジューシー」「複雑」と表現される
酸っぱさ・酢様（Sour / Acetic）欠点

食酢・過発酵フルーツのようなピリッと尖る酸。酢酸主体。甘さが伴わず、不快感が残る。

  • 原因①：未抽出（甘さ・苦味が乗らず酸だけ前に出る）
  • 原因②：精製不良・過発酵による豆側の欠点
  • 「酢っぽい」「ツン」「平坦に酸だけ」と感じる
Ch.3で動かすべきパラメータ まず豆の問題かを切り分ける：同じ豆を別の淹れ方で試して同じ尖りが出るなら豆側の欠点。抽出側の問題なら未抽出なので、 挽き目を1段細かく→湯温を2℃上げる→総時間を10〜20秒延ばすのうち1つだけ。甘さが上がってきたら正解の方向。
薄い（Weak / 低濃度）VS水っぽい（Watery / 未抽出）
バランスは取れているのに薄い vs バランスが崩れて空っぽ
▼
薄い（低濃度 / Low TDS）濃度の問題

味のバランスは取れている。酸・甘さ・苦味の関係性は正しいのに、ただ濃さが足りない。水で薄めたコーヒーのような印象。

  • 原因：レシオが薄い（湯量が多すぎ、または粉量が少なすぎ）
  • SCA基準ではTDS 1.15%未満
  • 飲んだ後の口の中は割と「正常」
水っぽい（未抽出 / Under-extracted）抽出率の問題

濃度の話ではなく抽出が手前で止まっている。酸ばかり前に出て甘さがない、後味が空っぽ。「あれ、なにこれ」という感じ。

  • 原因：挽き目が粗すぎ／湯温が低すぎ／時間が短すぎ
  • SCA基準ではEY 18%未満
  • 香りはあるのに味が乗っていない、という違和感
Ch.3で動かすべきパラメータ 「薄い」だけならレシオを濃く（1:16→1:15）で解決。「水っぽい」なら抽出率の問題なので挽き目を1段細かく→湯温を2℃上げる→時間を延ばす。両者の切り分けがCh.5で扱う「濃度 ≠ 抽出率」の本質。
Q
Q-Grader Tip: この3組の切り分けは、SCAカッピングの「Defects（欠点）」を素人向けに翻訳したもの。プロは Sour、Astringent、Bitter、Watery など1単語で原因を特定する語彙を持っている。あなたもこの3組さえ脳に入れば、自分のコーヒーが何で失敗しているかを10秒で判定できるようになる。判定 → Ch.3 でパラメータ調整 → 検証、というループが回り始める。
